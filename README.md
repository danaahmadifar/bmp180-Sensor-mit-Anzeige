# bmp180-Sensor-mit-Anzeige
In diesem kleinen Projekt wird auf 3 MAX7219 Matrizen die Temperatur oder die Altitude, gemessen von einem BMP180 Sensor, angezeigt. Ebenfalls werden die Werte der Temperatur, Luftdruck und Altitude parallel auf einem SSD1306 OLED Display parallel angezeigt. Mit einem Knopf kann man die Matrix-Anzeige zwischen °C/°F/Altitude wechseln.
